## List Users

Retrieve all registered users.

```
GET /v1/users
```

### Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number |
| `perPage` | `number` | `20` | Results per page (max 100) |
| `sort` | `string` | `-createdAt` | Sort field and direction |

### Response

```json
{
  "success": true,
  "data": {
    "users": [
      {
        "id": "abc123def456",
        "email": "user@example.com",
        "createdAt": "2026-03-08T00:00:00.000Z",
        "lastLogin": "2026-03-08T12:00:00.000Z",
        "loginCount": 24,
        "templateCount": 3,
        "successRate": 0.92
      }
    ],
    "totalItems": 42,
    "page": 1,
    "perPage": 20
  }
}
```

## Get User

Retrieve a specific user profile.

```
GET /v1/users/:id
```

```json
  "data": {
    "id": "abc123def456",
    "email": "user@example.com",

    "failedLogins": 2,
    "successRate": 0.92,
    "avgQualityScore": 0.78,
    "confidenceThreshold": 0.45,

```

## Delete User

Permanently delete a user and all their biometric data (face descriptors, templates, scan history).

```
DELETE /v1/users/:id
```

<Callout type="warn">
  This action is irreversible. All face descriptors, templates, and sign-in logs for this user will be permanently deleted. This is required for GDPR/BIPA compliance when a user requests data deletion.
</Callout>

### Response

```json
{
  "success": true,
  "data": {
    "deleted": true,
    "userId": "abc123def456",
    "deletedRecords": {
      "profile": 1,
      "templates": 3,
      "scans": 12,
      "signInLogs": 24
    }
  }
}
```

## Get Sign-In History

Retrieve sign-in logs for a specific user.

```
GET /v1/users/:id/logs
```

### Query Parameters

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | `number` | `1` | Page number |
| `perPage` | `number` | `20` | Results per page |
| `success` | `boolean` | *all* | Filter by success status |

### Response

```json
{
  "success": true,
  "data": {
    "logs": [
      {
        "id": "log_abc123",
        "success": true,
        "confidence": 0.73,
        "ipAddress": "203.0.113.42",
        "userAgent": "Mozilla/5.0...",
        "createdAt": "2026-03-08T12:00:00.000Z"
      }
    ],
    "totalItems": 24,
    "page": 1,
    "perPage": 20
  }
}
```
