## Prerequisites

- A modern browser with WebGL support (Chrome, Firefox, Safari, Edge)
- A device with a camera (webcam, phone camera, tablet)
- Node.js 18+ (for the JavaScript SDK)

## Installation

```bash
npm install @facesmash/sdk
```

## React (Recommended)

The fastest way to add face login to a React app:

```tsx
import { FaceSmashProvider, FaceLogin } from '@facesmash/sdk/react';

function App() {
  return (
    <FaceSmashProvider config={{ apiUrl: 'https://api.facesmash.app' }}>
      <LoginPage />
    </FaceSmashProvider>
  );
}

function LoginPage() {
  return (
    <FaceLogin
      onResult={(result) => {
        if (result.success) {
          console.log('Welcome back,', result.user.name);
          window.location.href = '/dashboard';
        } else {
          console.error('Login failed:', result.error);
        }
      }}
      className="w-full h-80 rounded-xl overflow-hidden"
    />
  );
}
```

That's it — the `<FaceSmashProvider>` loads the ML models automatically, and `<FaceLogin>` handles the webcam, face detection, capture, and authentication.

## Vanilla JavaScript

No framework? Use the core client directly:

```javascript
import { createFaceSmash } from '@facesmash/sdk';

const client = createFaceSmash({
  apiUrl: 'https://api.facesmash.app',
});

// Load ML models (do this once on page load)
await client.init((progress) => {
  console.log(`Loading: ${progress}%`);
});

// Capture images from webcam (base64 data URLs)
// Then authenticate:
const result = await client.login(images);

if (result.success) {
  console.log('Welcome back,', result.user.name);
} else {
  console.error(result.error);
}
```

See [Vanilla JS guide](/docs/sdk/vanilla-js) for a complete webcam capture example.

## Try It Now

Visit [facesmash.app](https://facesmash.app) to see FaceSmash in action:

1. Click **Register** and follow the prompts to capture your face
2. Enter your name and complete registration
3. Click **Sign In** and look at your camera — you'll be authenticated in seconds
