## Installation

```bash
npm install @facesmash/sdk
```

The core entry point (`@facesmash/sdk`) is framework-agnostic — no React or other framework required.

## Basic Usage

### Initialize

```javascript
import { createFaceSmash } from '@facesmash/sdk';

const client = createFaceSmash({
  apiUrl: 'https://api.facesmash.app',
  debug: true,
});

// Load ML models (call once on page load)
await client.init((progress) => {
  console.log(`Loading models: ${progress}%`);
});

console.log('Ready!', client.isReady);
```

### Login

Pass an array of base64 image data URLs captured from a webcam or file input:

```javascript
const result = await client.login(images);

if (result.success) {
  console.log('Welcome back,', result.user.name);
  console.log('Email:', result.user.email);
  console.log('Similarity:', result.similarity);
} else {
  console.error('Login failed:', result.error);
}
```

### Register a User

```javascript
const result = await client.register('Jane Doe', images, 'jane@example.com');

if (result.success) {
  console.log('Registered:', result.user.name);
  console.log('User ID:', result.user.id);
} else {
  console.error('Registration failed:', result.error);
  // e.g. "This face is already registered to John Doe"
}
```

### Analyze a Face

Get detailed quality metrics for a single image:

```javascript
const analysis = await client.analyzeFace(base64Image);

if (analysis) {
  console.log('Quality:', analysis.qualityScore);
  console.log('Confidence:', analysis.confidence);
  console.log('Lighting:', analysis.lightingScore);
  console.log('Head pose:', analysis.headPose); // { yaw, pitch, roll, isFrontal }
  console.log('Eye aspect ratio:', analysis.eyeAspectRatio);
  console.log('Rejection reason:', analysis.rejectionReason); // undefined if accepted
}
```

## Event System

Subscribe to SDK events for real-time feedback:

```javascript
const unsubscribe = client.on((event) => {
  switch (event.type) {
    case 'models-loading':
      console.log(`Loading: ${event.progress}%`);
      break;
    case 'models-loaded':
      console.log('Models ready');
      break;
    case 'face-detected':
      console.log('Quality:', event.analysis.qualityScore);
      break;
    case 'face-lost':
      console.log('No face detected');
      break;
    case 'login-success':
      console.log('Matched:', event.user.name, 'Similarity:', event.similarity);
      break;
    case 'login-failed':
      console.log('Failed:', event.error, 'Best:', event.bestSimilarity);
      break;
    case 'register-success':
      console.log('Registered:', event.user.name);
      break;
    case 'register-failed':
      console.log('Registration failed:', event.error);
      break;
  }
});

// Later: stop listening
unsubscribe();
```

## Low-Level API

For full control over the face recognition pipeline, import individual functions:

```javascript
import {
  loadModels,
  extractDescriptor,
  analyzeFace,
  calculateSimilarity,
  enhancedMatch,
  multiTemplateMatch,
} from '@facesmash/sdk';
```

### Extract a face descriptor

```javascript
const config = { modelUrl: 'https://cdn.jsdelivr.net/npm/@vladmandic/face-api/model', minDetectionConfidence: 0.3, debug: false };
await loadModels(config);

const descriptor = await extractDescriptor(base64Image, config);
// Float32Array(128) — the face embedding
```

### Compare two faces

```javascript
const similarity = calculateSimilarity(descriptor1, descriptor2);
// 0-1 score (1 = identical)

const match = enhancedMatch(descriptor1, descriptor2, 0.45, 0, 0.7);
// { isMatch: true, similarity: 0.72, adaptedThreshold: 0.45 }
```

### Multi-template matching

```javascript
const templates = [
  { descriptor: stored1, quality: 0.9, weight: 1.5 },
  { descriptor: stored2, quality: 0.7, weight: 1.0 },
];

const result = multiTemplateMatch(newDescriptor, templates, 0.45, 0.8);
// { isMatch: true, bestSimilarity: 0.78, avgSimilarity: 0.71, matchCount: 2 }
```

## TypeScript Support

The SDK is fully typed. Key types:

```typescript
interface FaceSmashConfig {
  apiUrl?: string;
  modelUrl?: string;
  matchThreshold?: number;
  minQualityScore?: number;
  minDetectionConfidence?: number;
  maxTemplatesPerUser?: number;
  debug?: boolean;
}

interface FaceAnalysis {
  descriptor: Float32Array;
  normalizedDescriptor: Float32Array;
  confidence: number;
  qualityScore: number;
  lightingScore: number;
  headPose: { yaw: number; pitch: number; roll: number; isFrontal: boolean };
  faceSizeCheck: { isValid: boolean; ratio: number; reason?: string };
  eyeAspectRatio: number;
  rejectionReason?: string;
}

interface LoginResult {
  success: boolean;
  user?: UserProfile;
  similarity?: number;
  error?: string;
}

interface RegisterResult {
  success: boolean;
  user?: UserProfile;
  error?: string;
}

interface MatchResult {
  isMatch: boolean;
  similarity: number;
  adaptedThreshold: number;
}

interface MultiTemplateMatchResult {
  isMatch: boolean;
  bestSimilarity: number;
  avgSimilarity: number;
  matchCount: number;
}
```

## Capturing Images from Webcam

The SDK expects base64 data URLs. Here's a minimal example of capturing from a webcam:

```javascript
async function captureFromWebcam(count = 3, delay = 500) {
  const stream = await navigator.mediaDevices.getUserMedia({
    video: { width: 640, height: 480, facingMode: 'user' },
  });

  const video = document.createElement('video');
  video.srcObject = stream;
  video.autoplay = true;
  await new Promise((r) => video.addEventListener('playing', r, { once: true }));

  const canvas = document.createElement('canvas');
  canvas.width = video.videoWidth;
  canvas.height = video.videoHeight;
  const ctx = canvas.getContext('2d');

  const images = [];
  for (let i = 0; i < count; i++) {
    ctx.drawImage(video, 0, 0);
    images.push(canvas.toDataURL('image/jpeg', 0.9));
    if (i < count - 1) await new Promise((r) => setTimeout(r, delay));
  }

  stream.getTracks().forEach((t) => t.stop());
  return images;
}

// Usage
const images = await captureFromWebcam();
const result = await client.login(images);
```
