## Overview

The FaceSmash SDK (`@facesmash/sdk`) provides everything you need to add facial recognition authentication to your web application. It runs entirely in the browser using WebGL-accelerated TensorFlow.js models.

```bash
npm install @facesmash/sdk
```

## Entry Points

| Import | Description |
|---|---|
| `@facesmash/sdk` | Core client — framework-agnostic, works with vanilla JS or any framework |
| `@facesmash/sdk/react` | React components and hooks — drop-in `<FaceLogin>`, `<FaceRegister>`, provider, and hooks |

## Core Capabilities

| Capability | Description |
|---|---|
| **Face Detection** | SSD MobileNet v1 primary detector with TinyFaceDetector fallback |
| **Descriptor Extraction** | 128-dimensional face descriptor via @vladmandic/face-api |
| **Quality Analysis** | Lighting, head pose, eye aspect ratio, and face size scoring |
| **Template Matching** | Multi-template weighted matching with adaptive thresholds |
| **Adaptive Learning** | Face embeddings improve over time with each successful login |

## Browser Requirements

- **WebGL** — Required for TensorFlow.js GPU acceleration
- **getUserMedia** — Camera access via the MediaDevices API

### Supported Browsers

| Browser | Min Version |
|---|---|
| Chrome | 80+ |
| Firefox | 78+ |
| Safari | 14+ |
| Edge | 80+ |

## Quick Example (React)

```tsx
import { FaceSmashProvider, FaceLogin } from '@facesmash/sdk/react';

function App() {
  return (
    <FaceSmashProvider config={{ apiUrl: 'https://api.facesmash.app' }}>
      <FaceLogin
        onResult={(result) => {
          if (result.success) {
            console.log('Welcome back,', result.user.name);
          } else {
            console.error('Login failed:', result.error);
          }
        }}
      />
    </FaceSmashProvider>
  );
}
```

## Quick Example (Vanilla JS)

```typescript
import { createFaceSmash } from '@facesmash/sdk';

const client = createFaceSmash({
  apiUrl: 'https://api.facesmash.app',
});

await client.init();

const result = await client.login(images);
// { success: true, user: { name: 'Jane Doe', email: '...' }, similarity: 0.78 }
```
