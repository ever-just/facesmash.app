## Installation

```bash
npm install @facesmash/sdk
```

React bindings are available from the `@facesmash/sdk/react` entry point. React 17+ is required as a peer dependency.

## `<FaceSmashProvider>`

Context provider that initializes the SDK client and loads ML models. Wrap your app (or the section that uses face auth) with this provider.

```tsx
import { FaceSmashProvider } from '@facesmash/sdk/react';

function App() {
  return (
    <FaceSmashProvider
      config={{ apiUrl: 'https://api.facesmash.app' }}
      onReady={() => console.log('Models loaded!')}
      onError={(err) => console.error(err)}
    >
      {/* Your face auth UI goes here */}
    </FaceSmashProvider>
  );
}
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `config` | `FaceSmashConfig` | `{}` | SDK configuration (see [Configuration](/docs/sdk/configuration)) |
| `onReady` | `() => void` | `undefined` | Called when ML models finish loading |
| `onError` | `(error: string) => void` | `undefined` | Called on model loading error |
| `onEvent` | `(event: FaceSmashEvent) => void` | `undefined` | Called for any SDK event |

## `<FaceLogin />`

Drop-in login component. Renders a webcam feed, auto-captures face images, and authenticates.

```tsx
import { FaceSmashProvider, FaceLogin } from '@facesmash/sdk/react';

function LoginPage() {
  return (
    <FaceSmashProvider config={{ apiUrl: 'https://api.facesmash.app' }}>
      <FaceLogin
        onResult={(result) => {
          if (result.success) {
            console.log('Welcome back,', result.user.name);
            window.location.href = '/dashboard';
          } else {
            console.error('Login failed:', result.error);
          }
        }}
        className="w-full h-80 rounded-xl overflow-hidden"
      />
    </FaceSmashProvider>
  );
}
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `onResult` | `(result: LoginResult) => void` | *required* | Called with the login result |
| `captureCount` | `number` | `3` | Number of images to capture for matching |
| `captureDelay` | `number` | `500` | Delay between captures in ms |
| `autoStart` | `boolean` | `true` | Auto-start scanning when component mounts |
| `className` | `string` | `undefined` | CSS class for the container |
| `overlay` | `ReactNode` | `undefined` | Custom overlay rendered on top of the video |
| `loadingContent` | `ReactNode` | `undefined` | Custom loading UI |
| `errorContent` | `(error: string, retry: () => void) => ReactNode` | `undefined` | Custom error UI |

### LoginResult

```typescript
interface LoginResult {
  success: boolean;
  user?: UserProfile;    // { id, name, email, face_embedding, created, updated }
  similarity?: number;   // 0-1 match score
  error?: string;
}
```

## `<FaceRegister />`

Drop-in registration component. Renders a webcam feed, captures face images, and registers a new user.

```tsx
import { FaceSmashProvider, FaceRegister } from '@facesmash/sdk/react';

function RegisterPage() {
  const [name, setName] = useState('');

  return (
    <FaceSmashProvider config={{ apiUrl: 'https://api.facesmash.app' }}>
      <input value={name} onChange={(e) => setName(e.target.value)} placeholder="Your name" />
      <FaceRegister
        name={name}
        email="user@example.com"
        onResult={(result) => {
          if (result.success) {
            console.log('Registered:', result.user.name);
          } else {
            console.error('Registration failed:', result.error);
          }
        }}
      />
    </FaceSmashProvider>
  );
}
```

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `name` | `string` | *required* | User's display name |
| `email` | `string` | `undefined` | Optional email (auto-generated if not provided) |
| `onResult` | `(result: RegisterResult) => void` | *required* | Called with the registration result |
| `captureCount` | `number` | `3` | Number of images to capture |
| `captureDelay` | `number` | `500` | Delay between captures in ms |
| `autoStart` | `boolean` | `true` | Auto-start when component mounts |
| `className` | `string` | `undefined` | CSS class for the container |
| `overlay` | `ReactNode` | `undefined` | Custom overlay on the video |
| `loadingContent` | `ReactNode` | `undefined` | Custom loading UI |
| `errorContent` | `(error: string, retry: () => void) => ReactNode` | `undefined` | Custom error UI |

## Hooks

### `useFaceSmash()`

Access the SDK client and loading state from anywhere within `<FaceSmashProvider>`.

```typescript
import { useFaceSmash } from '@facesmash/sdk/react';

const { client, isReady, isLoading, loadProgress, error, retryInit } = useFaceSmash();
```

| Return | Type | Description |
|---|---|---|
| `client` | `FaceSmashClient` | The SDK client instance |
| `isReady` | `boolean` | Whether ML models are loaded |
| `isLoading` | `boolean` | Whether models are currently loading |
| `loadProgress` | `number` | Loading progress (0-100) |
| `error` | `string \| null` | Error message if loading failed |
| `retryInit` | `() => void` | Retry model loading |

### `useFaceLogin()`

Programmatic login hook — call `login()` with base64 images.

```typescript
import { useFaceLogin } from '@facesmash/sdk/react';

const { login, isScanning, result, reset, isReady } = useFaceLogin();

// Trigger login with captured images
const loginResult = await login(base64Images);
```

### `useFaceRegister()`

Programmatic registration hook.

```typescript
import { useFaceRegister } from '@facesmash/sdk/react';

const { register, isRegistering, result, reset, isReady } = useFaceRegister();

const regResult = await register('Jane Doe', base64Images, 'jane@example.com');
```

### `useFaceAnalysis()`

Analyze a single face image for quality, lighting, and head pose.

```typescript
import { useFaceAnalysis } from '@facesmash/sdk/react';

const { analyze, analysis, isAnalyzing, isReady } = useFaceAnalysis();

const faceAnalysis = await analyze(base64Image);
// { qualityScore, confidence, lightingScore, headPose, eyeAspectRatio, ... }
```
